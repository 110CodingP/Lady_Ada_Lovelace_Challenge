So we have created a frontend! Let's add the power of backend now!

### Tasks
- Create a basic Express.js server listening at PORT 3000 which serves 3 opportunity cards 'GSoC 2026', 'PClub Secy Recruitment' and 'Google STEP' at endpoints '/gsoc', '/secy', '/step' endpoints.

- Create a basic frontend using the opportunity page you had created for the midterm i.e., in your frontend app (make it using vite and not expo) render Opportunity.jsx instead of App.js. In Opportunity.jsx have these 3 cards with data fetched from the above 3 api endpoints.
